Question 1: What is ASTER, and what is the satellite on which ASTER is onboard?

ASTER is an acronym for Advanced Spaceborne Thermal Emission and Reflection Radiometer. It is a Japanese sensor which is one of fiveremote sensory deviceson board theTerrasatellite launched intoEarth orbitbyNASAin 1999.

Question 4: What does NDVI measure, how is it calculated, and how does the ArcGIS NDVI function differ from the general NDVI formula?

NDVI measures the density of plant life on an area of land. The pigment in plant leaves, chlorophyll, strongly absorbs visible (red) light (from 0.4 to 0.7 m) for use in photosynthesis. The cell structure of the leaves, on the other hand, strongly reflects near-infrared light (from 0.7 to 1.1 m). NDVI exploits this phenomena using the following general formula:
NDVI = (NIR - VISR)/(NIR + VISR)
The ArcGIS NDVI formula was defined to better fit within an 8-bit data structure, which can easily be rendered with a specific color ramp or color map:
NDVI = ((NIR - VISR)/(NIR + VISR)) * 100 + 100


Question 5: What is the NDVI value range, and what do high values of NDVI mean?

NDVI has a value range of 0 to 200. The higher the value, the higher the density of plant life there is on the land being measured.

    Where NIR is the amount of near infrared and R is the amount of red light. With the ASTER data in this lab, you will use band 3 for NIR and use band 2 for R to compute NDVI.

    4. Use the following four lines of code to compute the pre-fire NDVI. The first two lines create a raster object each and assign it to a variable. Therefore, variable prered holds a raster object for band 2 and variable prenir holds a raster object for band 3 of the pre-fire dataset.

      Python: 
      import arcpy

prered = Raster("AST17042B2")
prenir = Raster("AST17042B3")
prendvi = (prenir * 1.0 - prered) / (prenir + prered)
prendvi.save("prendvi")

      The third line performs calculation. In Python, when an integer number is divided by an integer number, the result is an integer. For example, 5 / 3 results 1. Since cell values of both input rasters are integers, any arithmetic operation with these rasters would produce integer results. In order to obtain floating-point NDVI values, the first raster layer prenir is multiplied by a floating-point number 1.0 to make cell values floating point numbers a workaround that cheats Python.

      After executing the third line, an NDVI raster layer will be created and added into ArcMap. The raster is temporarily hold by variable prendvi. To permanently save the resulting NDVI raster, the last line calls the Save method of the raster object to save it as a name specified in the parentheses ("prendvi"). The name to save does not have to be same as the variable name, but it must be double quoted. After saving the raster, check your workspace to see if the raster exists (you may need to refresh the geodatabase).
    5. Follow example of the last four lines of code to calculate NDVI for the post-fire dataset and save the result raster as "postndvi" in your workspace. If there is any issue in displaying the resulting rasters, you may have to close ArcMap and reopen it.
